# MusicUtil

It is a python module for searching and downloading songs from various sources.

### Sources Supported:-
- [chiasenhac_vn](http://chiasenhac.vn)

### Upcoming Source Support:-
- [Spotify](https://spotify.com)
- [Youtube](https://youtube.com)

#### Requirements:-
* requests
* beautifulsoup4
* spotify
* html 5 lib
